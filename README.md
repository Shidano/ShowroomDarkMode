# ShowroomDarkMode
## SHOWROOM暗くするやつ  
### showroomをとりあえず暗くするためのchrome/Firefox用拡張機能です。  
スパゲティです。修正して個人利用すればいいと思うの。  
パッケージ、動かなくなるみたいだからパッケージ化されてない拡張機能を読み込む方法なら動くらしいよ。
